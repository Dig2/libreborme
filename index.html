Acerca de LibreBORME

LibreBORME es una plataforma web para la consulta y el análisis del Boletín Oficial del Registro Mercantil.

LibreBORME nace en 2014 como un Proyecto de Fin de Carrera con la intención de «abrir» los datos del Registro Mercantil de España. Aprovechando que desde 2009 se publica el BORME en formato electrónico, LibreBORME se encarga de leérselo por ti cada mañana y añadir a su base de datos los últimos cambios. De esta manera tú puedes hacer búsquedas semánticas, recibir notificaciones y usar estos datos de manera sencilla y para lo que quieras.
